Cutizone: Cuti Management Web App

Individual Class Assignment (Sep - Nov 2024)

My Role

  • UI Designer

Tools

  • Figma

Project Overview

Cutizone is a simple web-based platform concept designed to help employees submit and manage their leave requests more easily. The idea was born from a class assignment focusing on workplace challenges, and the project was developed individually.

Project Background

The brief given by our teacher was to address a common employee issue. I chose to focus on the hassle of manual leave submissions, which can be unclear, time-consuming, or disorganized in some workplaces.

Solution

A simple landing page that introduces the platform, along with a concept for an easy-to-use leave request form.

Project Status

This project was not further developed but served as an individual design exploration.
